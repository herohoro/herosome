# サブスクの整頓

お家大好きだし

お家を楽しく過ごせるサブスクは大好きさ。

だけどね

課金しすぎはよくない。

本当に必要なサブスクだけに絞って

浮いたお金はスタバ代にしたい。

ってことで

今日は私の今使ってるサブスクと

今後も課金する必要があるか否かについての１人会議の様子を紹介します📝

# 吟味：やりたいことに力をくれるサブスク

本来であれば時間を膨大にかけて習得していかなければならないような分野

ちょっとでも迷子にならないで目標を達成できるサブスクについて考えてみます。

AI関連のサブスクを増やしてしまい….

確かに便利なのですが、なくてもなんとかなる。

便利だからあえて余計なことをやってる感じも否めない。

- ChatGPT：20ドル
- ~~Copilot：10ドル~~
- NotionAI[年払い]：96ドル

コードを書くときに解説してもらわないと分からないことがたくさんあるので、

ChatGPTは必須。

NotionAIは英語に翻訳する機会が少しずつ増えてきそうな予感がしているので残す。

Copilotは….

確かにSuggestしてくれて便利だけど今の私のレベルでは手に余る感じがしてます。

これは元々コードを書ける人にとっては嬉しいけど、

知らないことばかりで解説を頼りに開発を進めるようなタイプには不要かもしれないなと思ってきました。

ゆくゆくはChatGPTを卒業してCopilot一本で行ければ節約できるけど。。。

まだまだ道は険しいですwww

険しいくても力を付けるオアシスとして

mosyaを買いました。

web制作に関する情報もmosyaのコミュニティの方でキャッチできるので

ひたすらネットサーフィンをして消耗せずに済むのが個人的に気に入っています♥

- mosya[買い切り]：14800円





AIでいくら翻訳が楽になるからとはいえ、

原文で理解できたり、ちょっとした会話はできた方がスムーズにことが進む…はず

塾のように宿題やったりテストをするほどのモチベーションは維持できないけれど

ちょっと片手間に数秒でも英語に触れる習慣を身につけたくって年払いで腹を決めました🔥

- Duolingo[年払い]：9000円
- Speak[年払い]：12800円

# 吟味：自分らしく過ごせるサブスク

忙しい生活は無理なので

ある程度無駄なことをやれる余白がないと

勉強も仕事もはかどりません。

- Netflix：990円
- Spotify：980円

BGMないとオシャレ感出ないし、

オシャレ感ないと勉強する気になれない。

疲れたときは

現実逃避がしたくって

韓国ドラマに浸りたくなる日がある。

誰とも遊び行かないんだからこれくらいはいいっしょ

# 吟味：自由時間を作るための時短サブスク

最近出会ったyoom

いろんなツールを連携させて自動処理してくれる。

GASとかGitHub Actionとか格闘しなくっても

工程ブロックを組み合わせて登録しておけば

あとは勝手にやってくれる。

今は、discordのコミュニティの作業通知やキュレーションのような機能を自動で処理してもらってます。

- Yoom[年払い]：16,364円

まだまだ使い始めたばかりなので

無意識にやっている無駄なことを洗い出して自動で処理できることをどんどんYoomにやってもらおうと思ってるところです。

工程ブロックを作ってて気づいたのが….

Notionからはじまり、Discordで終わるというパターン。

Notionでドキュメント管理をしているから

複数人で取り組むと進捗が追えず

いつも開いているDiscordに通知が来れば楽だ….という思考回路なんだなと気づきました。

# おわりに

結局１人会議をして分かったことは…

「Copilotはいらない」っていうことでしたw

開発意外にも使えるChatGPTはやっぱ万能だし

多分語学ネタでもお世話になりそうな予感がしています。

これだけ払ってるんだから結果出さなきゃもったいないなって思うので

夏バテしない程度に

進んで行こうと思います🔥🔥
